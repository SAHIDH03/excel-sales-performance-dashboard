# 📊 Excel Sales Performance Dashboard

## 🚀 Project Overview

This project is an interactive Excel dashboard built using Power Pivot and DAX to analyze market performance against sales targets.

The dashboard provides insights into:

* Sales trends
* Market-wise performance
* Variance analysis
* Growth percentage
* Top and bottom performing markets

---

## 🛠 Tools Used

* Microsoft Excel
* Power Pivot
* DAX Measures
* Pivot Tables
* Conditional Formatting
* Dashboard Visualization

---

## 📈 Key Features

* Dynamic KPI calculations
* Variance vs Target analysis
* Top 5 performing markets
* Bottom 5 underperforming markets
* Growth analysis with conditional formatting
* Executive summary dashboard


---

## 📌 Business Insights

* Overall market performance was below target by 10.4%
* Austria and Spain showed strong positive growth
* Pakistan and Poland required immediate attention due to underperformance
